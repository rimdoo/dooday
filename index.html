오늘 필요한 정보만 한 곳에

캘린더

macOS 캘린더 권한으로 오늘의 회의와 일정을 불러옵니다.

Jira

할당된 업무와 상태를 확인하고, 필요한 항목으로 바로 이동합니다.

Mail

Apple Mail에 연결된 계정의 읽지 않은 메일을 간단히 훑어봅니다.

local first

개인 데이터는 macOS 권한 안에서

Dooday는 캘린더와 메일을 별도 외부 API 대신 macOS 로컬 권한으로 읽습니다. Jira 연동만 OAuth 서버를 사용하며, 설정과 토큰은 현재 v1 기준 로컬 환경에 저장됩니다.
